# Section 03: File I/O

Real programs need to read and write data to files. Whether it's a simple text file, a CSV spreadsheet, or a JSON config, this section teaches you how to handle them all.

- Read from and write to text files safely
- Parse and create CSV files
- Work with JSON data (the language of the web)
- Handle file paths across different operating systems
